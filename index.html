Convolutional Neural Networks : What and Why

What: Special NN for processing grid like data

Features: 3 fundamental features for reducing the number of parameters.

1. Sparse interacitons between the layers => indirect interactions
2. Parameter sharing
3. Equivariant representaiton : f(g(x)) = g(f(x)) => consider f = translation and g = convolution. Equivariance of convolution w.r.t translaiton i usefull for parameter sharing acrosss the data. For example in image data the first convolution layer generally focuses on the edge dectection, however similar edgess may occur throughout the image, so it makes sense to represent them with same parameters.

Depth wise seperable Convolution

1. Depth wise convolution : Filtering stage. => Applies convolution to single channel at a time.
2. Point wise convolution : Combination stage.=> Performs linear combination on each of the previous layers.

AE vs VAE

The fundamental problem with autoencoders, for generation, is that the latent space they convert their inputs to and where their encoded vectors lie, may not be continuous, or allow easy interpolation.
Variational Autoencoders (VAEs) have one fundamentally unique property that separates them from vanilla autoencoders, and it is this property that makes them so useful for generative modeling: their latent spaces are, by design, continuous, allowing easy random sampling and interpolation.
